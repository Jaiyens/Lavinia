## (b) RATE SCHEDULE — decoded in plain English

- **Stored schedule (inventory):** `HAGA1`. The leading "H" is PG&E's
  hosting/territory prefix; the tariff is **AG-A1**.
- **Schedule as the bill prints it (`latestTariff`):** **"AGA1 Ag<35 kW Low Use"**.

**What AG-A1 is.** AG-A1 is PG&E's *current* small-agriculture time-of-use rate
for pumps whose maximum demand is **under 35 kW** AND whose *energy use is low*
(the "Low Use" / A1 branch; the high-use sibling is AG-A2). AG-A is the
sub-35 kW family; AG-B and AG-C are the 35-kW-and-above families. It is a
**current** (non-legacy) schedule, so it is NOT a candidate for the
"get-off-a-closed-legacy-rate" finding.

**Its price components (from the dated rate card `fixtures/pge-ag-rate-card.json`,
effective 2026-03-01):**

- **Customer charge:** **$0.68895 / meter / day** (≈ **$20.96 / month**). This is
  the fixed "keep the meter connected" charge and it is owed even when the pump
  never runs — which is exactly what this bill is (see below).
- **Demand charge (max demand):** **$12.33 / kW** of monthly billed maximum
  demand, the same rate summer and winter. (Card note: 12.33 is bill-sourced from
  the pre-03/01 AGA1 prints and may run slightly high.)
- **Energy ($/kWh), time-of-use:**
  - Summer (May-Oct): peak $0.40, partial-peak $0.37, off-peak $0.35 per kWh.
  - Winter (Nov-Apr): peak $0.33989, partial-peak $0.32500, off-peak $0.31060 per kWh.
- **TOU windows** (standard PG&E ag TOU; the card stores only the summer-month
  list `[5,6,7,8,9,10]`, not clock windows — windows below are the published ag
  definitions, surfaced for the farmer in plain terms): peak is the
  late-afternoon/evening block (roughly 4-9 pm), partial-peak brackets it, and
  off-peak is the overnight/midday remainder. Almond pumping that runs overnight
  lands almost entirely in off-peak.

**Why this meter is on AG-A1.** P048 is a small (<35 kW) ag pump. The bill itself
prints the "<35 kW Low Use" descriptor, and the meter shows **zero metered kWh**
this cycle, so PG&E's own "low use" small-ag branch (A1) is the correct home for
it. There is no demand history this cycle to push it toward A2 (high use) or into
the 35-kW+ B/C families.
