**What AG-A1 is, plainly:** AG-A1 is PG&E's agricultural rate for **small** services that draw
**under 35 kW** and run **light / occasional ("Low Use")**. Three pieces:

  vs AG-A2's $21.43/kW).

So AG-A1 is the right home for a meter that runs a little and rarely pulls big demand — it
trades higher per-kWh energy for a low demand charge.

**Why this meter is on AG-A1:** it is a **light-use** service — 145 kWh and a **3.734 kW** peak,
far under the 35 kW line — exactly the "Low Use" profile AG-A1 is built for. Inventory code
