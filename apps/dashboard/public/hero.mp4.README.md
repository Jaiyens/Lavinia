# Hero footage slot

Drop bright, sunlit farm footage here as `hero.mp4` (with an optional
`hero-poster.jpg`). The hero (`src/components/hero.tsx`) will layer it
full-bleed over the golden-hour gradient with only a soft light scrim.

Requirements:
- Bright, airy, sunny. Morning or golden-hour light over a field/orchard.
- Do NOT use the old dark dusk dusk footage. The hero stays light.
- Until this file exists, the hero renders the `.gradient-hero` mesh alone.
